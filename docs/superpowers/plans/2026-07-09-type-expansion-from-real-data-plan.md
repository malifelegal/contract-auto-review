# 유형 세분화 계획 — 실제 검토리스트(6,631건) 기반

> 회사 계약검토 리스트(efefe.xlsx, 6,631건, 중분류11·소분류44) 분석 근거. 유형은 굵게(3개 추가=총8개), 세부는 자동 마킹 태그 + detect_keywords 보강.

## 데이터 분석 요지

- 회사 분류는 **부서/업무 축**(자산운용>투자및대출 등)이지 **법적 성격 축**이 아님 → 빈도 근거로만 활용.
- 계약명 6,631건 법적 성격 빈도(중복 가능): 투자768·위탁698·제휴640·SI602·대출담보495·용역467·구매389·임대차284·보증269·NDA137·주주간110·재보험103·라이선스85.
- 기존 유형(업무위탁·IT외주·용역구매·NDA·주주간) 미커버 최대 공백: **대출·담보(495)·제휴MOU(640)·투자(768)**.

## 결정: 3개 유형 추가 (총 8개)

1. **finance (금융·여신)** — 대출·융자·담보·ABL·이행보증. 보험사 자산운용 핵심. 495+269건.
   - 근거 법령 후보: 민법(금전소비대차 §598~, 담보물권 §329·§356·저당 §356, 보증 §428), 이자제한법, 보험업법(자산운용 §104~§109, 특별계정), 여신전문금융업법(해당 시), 동산·채권담보법. 이행보증=보증보험(상법 §726의5 등).
2. **alliance (제휴·업무협약)** — 제휴계약·MOU·업무협약. 640건. 마케팅지원·GA·퇴직연금에 다수.
   - 근거 법령 후보: 민법(위임·도급 혼합), 공정거래법(부당공동행위 §40·거래상지위남용 §45), 개인정보보호법(제휴 시 정보제공·위수탁 §17·§26), 보험업법(모집위탁·제3자 업무제휴), 표시광고법(공동마케팅).
3. **investment (투자)** — 펀드·출자·LP·voting agreement·조합. 768건. 자산운용.
   - 근거 법령 후보: 자본시장법(집합투자 §9⑱·§229, 사모 §249, 업무집행조합원, 투자일임/자문), 상법(합자조합 §86의2~, 익명조합 §78~), 보험업법 자산운용 규제, 민법 조합(§703~).

## 자동 마킹 태그(세부 — 유형 아님)

회사 소분류/계약명에서 자동 감지되는 태그. 유형 선택 부담 없이 세부 성격 표시. 기존 모듈(M-*) 자동감지 구조 확장.
- 태그 후보: [임대차/렌탈], [재보험], [라이선스/SW], [이행보증], [담보], [MOU], [해외/영문], [변경/합의서](기존계약 수정건 다수).
- 계약명 정규식으로 감지 → UI에 회색 보조 배지. 유형과 독립.

## detect_keywords 보강

- 6,631 계약명을 유형별 학습셋으로. 각 유형 detect_keywords를 실제 표현으로 확장(예: finance←대출약정·담보·ABL·리파이낸싱·여신·근질권).
- 검증: 계약명 샘플로 detectType 정확도 측정(회사 분류와 대조는 참고만 — 축이 다름).

## 구현 순서(제안)

1. **detect_keywords 보강 + 자동 마킹 태그** (지식 최소, 효과 즉시) — 유형 추가 전 먼저 감지·태깅.
2. **finance 유형** (최대 공백·보험사 핵심) — 법령 원문 조달(korean-law MCP)→스냅샷→체크리스트 큐레이션.
3. **alliance 유형**.
4. **investment 유형**.
5. 각 유형 seed 검수(quote↔원문)로 verified 승급.

## 법무 하네스 준수

- 모든 근거 quote는 DB 원문/korean-law MCP 원문에서 발췌, verified:false로 시작.
- severity는 norm_type 규칙 도출(강행→필수 등) + severity_basis.
- 새 유형 법령은 상당수 DB 미수록 예상 → external_laws.json seed + extract_snapshot.

## 미결/판단 필요

- finance 유형에 보증보험(상법)·이행보증을 별도 태그로 뺄지 유형 내 모듈로 둘지.
- investment의 조합/펀드가 주주간계약과 겹치는 영역(voting agreement) 경계.
- 재보험(103건)은 보험사 특수 — 별도 유형으로 뺄지 alliance 하위로 둘지(후속).
